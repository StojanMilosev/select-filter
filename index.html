Borders

The CSS border properties allow you to specify the style, width, and color of an element's border

Margins

The CSS margin properties are used to generate space around elements, the properties set the size of the white space outside the border.

Padding

The CSS padding properties are used to generate space around conten; clears an area around the content (inside the border) of an element.

Headings

Headings are defined with the h1 to h6 tags. h1 defines the most important heading. h6 defines the least important heading.

Paragraph

The HTML p element defines a paragraph

Input

The input element is the most important form element. The input element can be displayed in several ways, depending on the type attribute.

Variables

JavaScript Variables JavaScript variables are containers for storing data values.

Functions

A JavaScript function is a block of code designed to perform a particular task. A function is executed when "something" invokes it (calls it).

Objects

Objects are variables too. But objects can contain many values. The values are written as name:value pairs (name and value separated by a colon).
